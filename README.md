# 백링크 자동 글쓰기

그누보드 기반 사이트에 자동으로 글을 작성하는 GUI 프로그램

## 설치 방법

### 1. 필수 프로그램 설치
- Python 3.x
- Chrome 브라우저

### 2. 의존성 설치
```bash
pip3 install selenium
```

### 3. 실행
```bash
python3 backlink_auto_writer.py
```

## 기능
- 구글시트에서 URL 목록 및 프리셋 로드
- 원클릭 자동 입력 및 등록
- 모든 프리셋 일괄 등록
- 캡차 없는 사이트 전체 자동 등록
- 제목/내용 클립보드 복사
